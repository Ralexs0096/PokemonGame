# Pokemon Game Vue

Vue App using Poke-Api

## Run Locally

Clone the project

```bash
  git clone https://github.com/Ralexs0096/PokemonGame
```

Go to the project directory

```bash
  cd pokemonGame
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```
